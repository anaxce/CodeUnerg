¿Que es HTML?

HTML es un lenguaje de maquetado para describir documentos web (páginas web)

  • HTML significa Hyper Text Markup Language (Lenguaje de maquetado de hipertexto). HTML añade "Maquetado" al texto estándar en inglés. "Hipertexto" se refiere a enlaces - hipervínculos - que conectan páginas web entre sí



  • Un lenguaje de maquetado es un conjunto de etiquetas de maquetado valga la redundancia

  • Los documentos HTML se describen mediante etiquetas HTML

  • Cada etiqueta HTML describe contenido de documento diferente



  • HTML no distingue entre mayúsculas y minúsculas

Historia de HTML

  • HTML 2.OSe simplificaba al máximo la estructura del documento para agilizar su edición, donde la declaración explícita de los elementos body, html y head es opcional.

  • HTML 3.0 La versión HTML 3.2 se publicó en 1997 y es la primera recomendación de HTML publicada por el W3C (Consorcio internacional). Esta revisión incorporó los últimos avances de las páginas web desarrolladas hasta 1996



  • HTML 4.0 La última especificación oficial de HTML se publicó en diciembre de 1999 y se denomina HTML 4.01. las empresas Apple, Mozilla y Opera mostraron su preocupación por la falta de interés del W3C en HTML y decidieron organizarse en una nueva asociación llamada WHATWG (Web Hypertext Application Technology Working Group). en marzo de 2007 el W3C decidió retomar la actividad estandarizadora de HTML, dentro del cual decidió integrar el XHTML.

  • HTML 5 El consorcio internacional W3C, después de una evolución de varios años, liberó el HTML 5 como estándar oficial a finales de octubre de 2014. Hay nuevas etiquetas, se introduce la posibilidad de introducir audio y video de forma directa en la web sin necesidad de plugins o complementos en los navegadores, y otras novedades.
