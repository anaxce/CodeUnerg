HTML Etiquetas, Atributos y Elementos

Etiquetas

Las etiquetas html son palabras clave (nombres de etiqueta) rodeadas por corchetes angulares

content
  • Las etiquetas HTML normalmente vienen en parejas como y

  • La primera etiqueta es la etiqueta de inicio la que abre; La segunda etiqueta es la etiqueta final,la que cierra

  • La etiqueta de fin se escribe como la etiqueta de inicio, pero con una barra diagonal antes del nombre de la etiqueta, como vieron en el ejemplo de arriba.

Atributos

Las etiquetas también pueden tener atributos, que son bits adicionales de información. Los atributos aparecen dentro de la etiqueta de apertura y sus valores se sitúan entre comillas. Parecen algo así como:

< tag attribute= "value" > Flor
