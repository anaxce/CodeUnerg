Operadores Logicos

Los Operadores Logicos son usados en expresiones condicionales que retornan verdadero(TRUE) o falso (FALSE)

Verdadero = True
Cierto = True
Operador
Nombre
Ejemplo
Descripción
and
Y
$a and $b
Si ambos son ciertos, $a and $b son verdadero
or
O(Inclusivo)
$a or $b
Si uno es verdadero, $a or $b son verdadero
xor
O(Exclusivo)
$a xor $b
Es verdadero si $a o $b son ciertas, pero no ambas
!
Not(no)
!$a
Es verdadero si $a no es verdadero
&&
And (y)
$a && $b
Es verdadero, Si tanto $a como $b son verdadero
||
Or (o inclusivo)
$a && $b
Es verdadero si cualquiera de $a o $b es verdadera

La razón para tener las dos variaciones diferentes de los operadores "and" y "or" es que ellos operan con precedencias diferentes.(Vease Precedencia de operadores)

Ejemplo $x = 100 ;
$y = 80 ;

if ( $x == 100 and $y == 50 )
echo "Operador and";

if ( $x == 100 or $x == 80 )
echo "Operador or (o)";

if ( $x == 100 xor $x == 80 )
echo "Operador xor (o)";

if ( $x == 100 && $x == 50 )
echo "Operador && (and, y)";

if ( $x == 100 || $x == 80 )
echo "Operador || (Or, o)";

if($x !== 90)
echo "Operador != (Not)";
