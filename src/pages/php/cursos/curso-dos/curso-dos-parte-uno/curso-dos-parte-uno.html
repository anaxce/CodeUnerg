Operadores de Comparación

Los Operadores de Comparación en PHP son los siguientes

Operador
Nombre
Retorna TRUE, si se cumplen estas condiciones
$a == $b
Igual
si $a es igual a $b después de la manipulación de tipos.
$a != $b
Diferente
si $a no es igual a $b después de la manipulación de tipos.
< >
$a Diferente $b
Sinonimo de !=
$a !== $b
No Identico
si $a no es igual a $b, o si no son del mismo tipo.
$a < $b
Menor que
si $a es estrictamente menor que $b.
$a > $b
Mayor que
si $a es estrictamente mayor que $b.
$a <= $b
Menor o igual que
si $a es menor o igual que $b.
$a >= $b
Mayor o igual que
si $a es mayor o igual que $b.
$a <=> $b
Nave espacial
Un integer menor que, igual a, o mayor que cero cuando $a es respectivamente menor que, igual a, o mayor que $b. (Disponible a partir de PHP 7).
$a ?? $b ?? $c
Fusión de null
El primer operando de izquierda a derecha que exista y no sea NULL. NULL si no hay valores definidos y no son NULL. (Disponible a partir de PHP 7).

/* Esto sería ejecutado porque != Significa que la variable no es igual a la variable condicional */
if ( $x > 8 )
