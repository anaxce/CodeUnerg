Operadores de Asignación

El " = " es usado para asignar un valor a la variable.

Ejemplo $num1 = 3 ;
$num1 += $num1 ;
echo $num1 . "< br>";
$num2 = 7 ;
$num2 *= $num2 ;
echo $num2 ;
 Salida en la web 6
49

Explicación

La variable $num1, se añade a sí misma, lo que da 6. el $num2, se multiplica por sí mismo, lo que da 49.

Ejemplo $num1 = 3 ;
