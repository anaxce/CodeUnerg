Operadores para Arrays

Los Operadores de Arrays(Matrices) son usados para comparar Arrays.

Operador
Nombre
Ejemplo
Descripción
+
Unión
$a + $b
Union entre $a y $b
==
Igualdad
$a == $b
Devuelve verdadero si $a y $b tienen las mismas parejas clave/valor.
===
Identidad
$a === $b
Devuelve verdadero si $a y $b tienen las mismas parejas clave/valor en el mismo orden y de los mismos tipos
!=
Desigualdad
$a != $b
Devuelve verdadero si $a no es igual a $b
< >
Desigualdad
$a != $b
Devuelve verdadero si $a no es igual a $b
!==
No-Identidad
$a !== $b
Devuelve verdadero si $a no es identica a $b

El operador + devuelve el array del lado derecho añadido al array del lado izquierdo; para las claves que existan en ambos arrays, serán utilizados los elementos del array de la izquierda y serán ignorados los elementos correspondientes del array de la derecha.

Ejemplo $x = array (
"a" => "php",
"b" => "java"
); $y = array (
"c" => "html",
"d" => "javascript"
);
// Union
print_r ( $x+$y);
// Igualdad
var_dump ( $x==$y);
// Identidad
var_dump ( $x===$y);
// Desigualdad
var_dump ( $x!=$y);
// Desigualdad
var_dump ( $x < > $y);
// No Identidad
var_dump ( $x!==$y);
