Creando una Cookie

Una cookie, tambien conocida como HTTP cookie, web cookie, o browser cookie, Es una pequeña pieza de datos enviados desde un sitio web y almacenados en el navegador web de un usuario mientras el usuario está navegando en ese sitio web

Cada vez que el usuario carga el sitio web, el navegador envía la cookie de vuelta al servidor para notificar al sitio web de la actividad anterior del usuario

Creando una Cookie

Para crear una Cookie, se utiliza la funcion setcookie()

Sintaxis

setcookie(name, value, expire, path, domain, security);

Debe especificar tres argumentos, los otros son opcionales.

Esos argumentos son setcookie(name,value,expiration)

  • 1. name: El nombre de tu cookie Este nombre lo utilizaras mas tarde para recuperar tu cookie, ¡asi que no la olvides!

  • 2. value: El valor que se almacena en su cookie, los valores mas comunes son username(string) y ultima visita(date)

  • 3. expiration: La fecha en que la cookie expirará y se eliminará. Si no establece esta fecha de caducidad, se tratará como una cookie de sesión y se eliminará cuando se reinicie el explorador.
$cookie_name = "Daryl";
$cookie_value = "Dixon";
setcookie ($cookie_name, $cookie_value, time() + (86400 * 30), "/");
// 86400 = 1 day

Recuperar una Cookie

Para recuperar una Cookie se utiliza $_COOKIE

Para saber si la Cookie esta configurara o no, utilizamos isset() function

if(!isset($_COOKIE [$cookie_name]))

echo "Nombre de la Cookie '" . $cookie_name . "' No esta establecido! ";

else

echo "Cookie '" . $cookie_name . "' Esta establecido! /n";
echo " Valor es '" . $_COOKIE[$cookie_name] ;
