Variables

Las variables son contenedores (Espacios en la memoria) que contienen datos. Estos datos tambien pueden ser cambiados o modificados.

Reglas para declarar variables

  • Una variable debe comenzar con el signo "$" seguido del nombre de la variable.

  • El nombre de la variable debe comenzar con una letra o el carácter de subrayado "_"

  • El nombre de la variable no puede comenzar con un numero

  • El nombre de la variable sólo puede contener caracteres alpha-numeric y subrayados (A-z, 0-9 y _)

  • Los nombres de variable son sensibles a mayúsculas y minúsculas ($edad y $EDAD son dos variables diferentes)
Ejemplo: $myName = "Nestor";
echo = $myName;
