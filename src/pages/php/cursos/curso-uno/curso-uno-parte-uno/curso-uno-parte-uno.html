Nombrando una variable

Darle un buen correcto nombre a tus variables, funciones y clases siempre es muy importante.

Si define las siguientes variables.

$programming_language = "PHP";
$menu_items = 7;

$programming_language, por ejemplo no es un buen nombre, es demasiado largo. Un mejor nombre seria quizas $prg_language,ya que es mas corto y entendible.

//El lenguaje de programacion utilizado para escribir este script.
$prg_language = "PHP";
//El maximo numero de items permitidos en mi menu personal.
$menu_items = 7;
