Conectar a MySQL

Que es MySQL

MySQL Permite a los usuarios crear tablas, donde los datos se pueden almacenar mucho más eficientemente que la forma en que los datos se almacenan en las matrices.

Con el fin de utilizar MySQL o alguna otra bases de datos en general con eficacia, es necesario comprender SQL, o Query estructurado.

Estas guias estan basadas en funciones Mysqli y no las funciones antiguas de mysql

Paso a Paso

1. Conectando al servidor MySQL

Php tiene la función mysqli para conectarse a un servidor MySQL que maneja todo el manejo de socket de bajo nivel.

Hay 3 argumentos que utilizaremos mucho

  • 1. Nombre del servidor MySQL
  • 2. Nombre de usuario del MySQL
  • 3. Contraseña del MySQL

Veamos un ejemplo

$servername = "localhost" ;
$username = "username" ;
$password = "password" ;

// Creando conexión
$cn = new mysqli ($servername, $username, $password);

// Verificando Conexión
if ($cn->connect_error)

echo "Conexion Fallida : " . $cn->connect_error;

echo"Conexión Exitosa";

Cerrar la conexión

Para cerrar la conexion use lo siguiente

$cn->close();
