Creando consulta de base de datos

La sentencia CREATE DATABASE creará la base de datos en MySQL

La creación de la base de datos es nuestro primer paso al acceder al servidor MySql backend con un script php

Esto se puede lograr conectando con el servidor. Y luego crear la base de datos.

//Creando conexión
$cn = new mysqli( "localhost", "your_username", "my_password" );

//Verificando Conexion
if($cn->connect_error)

echo"Error estableciendo la conexion". $cn->connect_error;


//Creando bd
$sql = "CREATE DATABASE MyDB";

if($cn->query($sql) === TRUE)

echo "Base de datos creada";

else

echo "Error creando la base de datos" . $cn->error;

$cn->close();

Creando la tabla

El proceso de creación de la tabla es tan fácil como crear la base de datos. Tenemos que ejecutar la consulta create table usando el comando mysqli construct

//Creando conexión
$cn = new mysqli (
"localhost",
"my_username",
"my_password",
"MyDatabase"
);

//Verificando conexión
if($cn->connect_error)
echo "Conexión fallida : " . $cn->connect_error;


//sql para crear la tabla
$sql = "CREATE TABLE MyTable(nombre VARCHAR(18), apellido VARCHAR(18), salario DECIMAL(5,4) )";

if($cn->query($sql) === TRUE)

echo "Tabla creada";

else

echo "Error al crear la tabla : " . $cn->error;

$cn->close();
