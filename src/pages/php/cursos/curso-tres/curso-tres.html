Declaración If

las sentencias contienen declaraciones que sólo se ejecutan, cuando la condición se cumple

Ejemplo: $name = "Jon Snow";
if($name == "Jon Snow")

echo"Funcionó"(El Rey en el norte);
 Salida en la web "Funcionó"(El Rey en el norte)

Explicación

La declaración If simplemente dice, Si la variable $name es igual a "Jon Snow", se ejecuta la sentencia dentro de los corchetes. Si su variable tenía cualquier otro valor entonces las declaraciones dentro de los corchetes no se habria ejecutado. ¿Sencillo no? vamos a entenderlo poco a poco.


If, else if, and else:

elseif, como su nombre lo sugiere, es una combinación de if y else. Del mismo modo que else, extiende una sentencia if para ejecutar una sentencia diferente en caso que la expresión if original se evalúe como FALSE. Sin embargo, a diferencia de else, esa expresión alternativa sólo se ejecutará si la expresión condicional del elseif se evalúa como TRUE.

Ejemplo: $rank = 65;
if($rank <= 100 and $rank >= 75)

echo "Rango Diamante";

elseif ( $rank < 75 and $rank >= 60)

echo "Rango Platino";

elseif ( $rank < 60 and $rank >= 45)

echo "Rango Oro";

else

echo "Rango Plata";

 Salida en la web
Rango Platino
