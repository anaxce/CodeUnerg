For Loop

El Ciclo for Se utiliza cuando se sabe de antemano cuántas veces se debe ejecutar el script.

Sintaxis
for (inicialización; condición; incremento/decremento)

// Código a ejecutar ;

Veamos un ejemplo, ejecutaremos un pequeño programa usando el ciclo for que nos muestre los numeros del 1 al 10.

Ejemplo for( $i = 1; $i <= 10;$i++)

echo $i;
 Salida en la web
12345678910

Dentro del ciclo for tenemos la sentencia $i = 1; que declara una variable $i y lo establece igual a 1. A continuación, tenemos la condición $i <= 10 Que es la parte if de la declaración.

La condición dice que si $ i <= 10 se ejecutan las sentencias en los corchetes. Finalmente, tenemos una declaración de operador de incremento, $i++ Que se utiliza para avanzar el ciclo for.

Ya que el ++ es después de la variable, nosotros incrementaremos i despues de todas las declaraciones en los corchetes hayan sido ejecutadas.
