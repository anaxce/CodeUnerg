Foreach Loop

El constructor foreach proporciona un modo sencillo de iterar sobre arrays. foreach funciona sólo sobre arrays y objetos, y emitirá un error al intentar usarlo con una variable de un tipo diferente de datos o una variable no inicializada.

Existen dos sintaxis

Sintaxis 1
foreach (expresión_array as $valor)
sentencias

 Sintaxis 2
foreach (expresión_array as $clave => $valor)
sentencias

Los ciclos foreach son útiles cuando se trata de una matriz indexada con claves arbitrarias (no numéricas)

Ejemplo
$array = array(
"1st" => "Casa Stark",
"2nd" => "Casa Lannister" ,
"3rd" => "Casa Targaryen"
);
foreach ( $array as $someVar)

echo $someVar . < br>;
 Salida en la web
Casa Stark
Casa Lannister
Casa Targaryen

Foreach con valores clave

Si necesita usar las claves de la matriz en el cuerpo de su bucle, simplemente añada la variable como en la siguiente sentencia. La frase $myKey => $value hace que el valor de la clave sea accesible.

Ejemplo
$array = array(
"1st" => "Casa Stark",
"2nd" => "Casa Lannister" ,
"3rd" => "Casa Targaryen"
);
foreach ( $array as $key => $value)

echo $key . " : " . $value . "< br>" ;
 Salida en la web
1st: Casa Stark
2nd: Casa Lannister
3rd: Casa Targaryen
