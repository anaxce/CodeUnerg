While Loop

El Ciclo While es una instrucción de flujo de control que permite que el código se ejecute repetidamente en base a una condición booleana dada

Es mas facil de lo que suena, veamos su sintaxis primero.

Sintaxis
While (condición)

// Código a ejecutar ;
 Ejemplo
$i = 0 ;
while ( $i < 5 )

// Escriba el valor de la variable
echo $i . "< br>" ;
operador de incremento
$i++;
echo "Final del ciclo While";
 Salida en la web
0
1
2
3
4
End of while loop

Mientras la condición sea verdadera, se ejecutan las instrucciones contenidas en el while. Cuando deja de cumplirse la condición, se sale del ciclo y se continúa ejecutando el resto del programa.

Si la condición no se cumple ni la primera vez que se comprueba, las líneas en el interior del while no se ejecutarán ni una vez.
