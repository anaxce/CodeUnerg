Array

Una matriz (Array) es una variable, que puede contener más de un valor a la vez

  • Los arrays pueden almacenar cadenas, enteros o incluso otras matrices en un formato de tipo lista

  • Los arrays pueden ser accedidos usando el índice o el valor dominante y el índice del array comienza a partir de 0

  • Los arrays se utilizan a menudo siempre que haya x cantidad desconocida de variables que necesitan almacenarse para su posterior recuperación

¿Como definimos un array?

Sintaxis // Definiendo numeros enteros en la matriz
$numList = array(1,3,7);

//Accediendo al elemento 0 de la matriz
echo($numList[0] . "< br>");

//Accediendo al segundo elemento de la matriz
echo($numList[2]);
 Salida en la web 1
7

Funciones comunes en Arrays de php

Las funciones de Arrays son las siguientes:

1. array_pop() - Elimina el ultimo elemento

Ejemplo $pelis = array ("Los Vengadores", "Rapido y Furioso", "Interestellar");
array_pop($pelis);
print_r($pelis);
 Salida en la web Array
(
[0] => Los Vengadores
[1] => Rapido y Furioso
)

2. array_push() - Añade un elemento al final

Ejemplo $names = array ("Clark", "Bruce", "Diana");
array_push($names,"Barry");
print_r($names);
 Salida en la web Array
(
[0] => Clark
[1] => Bruce
[2] => Diana
[3] => Barry
)

3. array_search() - Busca un valor determinado en un array y devuelve la primera clave correspondiente en caso de éxito

Ejemplo $series = array ("The Walking Dead", "Game of Trhones", "Daredevil");
echoarray_search("Daredevil",$series);
 Salida en la web 2

4. array_shift() - Quita un elemento del principio del array

Ejemplo $dArray = array ("a", "b", "c");
echo array_shift($dArray);
print_r($dArray);
 Salida en la web (
[0] => b
[1] => c
)
