Array

Una matriz (Array) es una variable, que puede contener más de un valor a la vez

  • Los arrays pueden almacenar cadenas, enteros o incluso otras matrices en un formato de tipo lista

  • Los arrays pueden ser accedidos usando el índice o el valor dominante y el índice del array comienza a partir de 0

  • Los arrays se utilizan a menudo siempre que haya x cantidad desconocida de variables que necesitan almacenarse para su posterior recuperación

¿Como definimos un array?

Sintaxis // Definiendo numeros enteros en la matriz
$numList = array(1,3,7);

//Accediendo al elemento 0 de la matriz
echo($numList[0] . "< br>");

//Accediendo al segundo elemento de la matriz
echo($numList[2]);
 Salida en la web 1
7
