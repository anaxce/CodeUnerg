Funciones

Las funciones son extremadamente útiles, nos pueden ahorrar bastante tiempo y ayudarnos a no escribir tanto codigo redundante

Una función no es mas que un bloque de sentencias, que se utiliza para realizar una tarea específica.

Veamos un ejemplo de como es la sintaxis para crear una función

Creando una función function functionNombre()

// Codigo que se quiere ejecutar

Ahora vamos a crear y llamar una función

//Definiendo una function function vengadores()

echo "Vengadores";


//haciendo el llamado a la función
vengadores(); Salida en la web
Vengadores

Explicación

Aquí, tenemos una función llamada vengadores. Despues de definir la función, llamamos a la funcion con función_nombre vengadores();


Funciones con argumentos

Ejemplo 1

//$name es un argumento function msjBienvenida($name)

echo "Bienvenido a la guia de PHP, " . $name;


//haciendo el llamado a la función
msjBienvenida(" Batman");
 Salida en la web
Bienvenido a la guia de PHP, Batman

Un argumento es como una variable de la función que se define cuando se llama a la función más tarde.

Ejemplo 2

function addFunction($argumento1, $argumento2)

echo $argumento1 + $argumento2;


//haciendo el llamado a la función
addFunction(423,541);
 Salida en la web
964

Explicación

Llamamos a la función addFunction(). Pasa dos argumentos esta vez: 423 and 541. Donde 423 esta siendo pasado dentro de addFunction como $argumento1 y 541 se pasa tambien en $argumento2 Una vez que ambos son pasados, hacemos un echo de la suma de los dos parametros. en el resultado, puedes ver que al hacer echo nos devuelve 964
