Iniciar Sesión

¿Que es una Sesión?

No, no es una sesión al psicologo, Una sesión en PHP funciona para almacenar informacion de un usuario en el servidor para posteriormente ser utilizada(username, shopping cart items, etc).

Sin embargo, esta información de sesión es temporal y suele eliminarse muy rápidamente después de que el usuario ha dejado el sitio web que utiliza sesiones.

Las variables de sesión contienen información sobre un único usuario y están disponibles para todas las páginas de una aplicación

Fin de sesiones

Cuando el usuario cierra el navegador o cuando el servidor web elimina la información de la sesión o cuando el programador explícitamente destruye la sesión

Iniciar una sesión

Antes de almacenar la información del usuario en la sesión de PHP, primero debe iniciar la sesión. Cuando inicia una sesión, debe estar al principio de su código, antes de enviar cualquier codigo HTML o texto

session_start();

Las variables de sesión se establecen con la variable global de PHP $_SESSION


//Iniciar la sesión
session_start();
?>
< !DOCTYPE html>
< html>
< body>


// Variables de session
$_SESSION["nombre"] = "Daenerys";
$_SESSION["apellido"] = "Targaryen";
echo " Se establecieron las variables ";
?>
< /body>
< /html>
