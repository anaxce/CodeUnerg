Obtener valores de la variable de sesión

Crear otra página php. Desde esta página, accederemos a la información de sesión que establecimos en la primera página.


//Iniciar la sesión
session_start();
?>
< !DOCTYPE html>
< html>
< body>


// haremos un echo a las variables de sesión que se establecieron en la página anterior
echo"Nombre : " . $_SESSION[ "nombre"];
echo"Apellido : " . $_SESSION[ "apellido"];
?>
< /body>
< /html>

Modificar una variable de sesión php

Modificar una variable de sesión, nuevamente


//Iniciar la sesión
session_start();
?>
< !DOCTYPE html>
< html>
< body>


// para cambiar una variable de sesión, solo escribalo
$_SESSION["nombre"] = "Jorfrey";
print_r($_SESSION);
?>
< /body>
< /html>

Destruir una sesión

Con el fin de eliminar todas las variables de sesión global y destruir la sesión, usamos session_unset() y session_destroy


//Iniciar la sesión
session_start();
?>
< !DOCTYPE html>
< html>
< body>


// Remover todas las variables de sesión
session_unset();
// Destruir la sesión
session_destroy();
?>
< /body>
< /html>
