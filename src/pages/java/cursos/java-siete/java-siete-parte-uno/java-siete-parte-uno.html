Constructor parametrizado

En este ejemplo, hemos creado el constructor de la clase 'Estudiantes' que tiene dos parámetros. Podemos tener cualquier número de parámetros en el constructor.

class Estudiantes

int id;
String nombre;

Estudiantes (int i, String n)

id = i;
name = n;


void display()

System.out.println(id+ " " +name);

public static void main(String args[ ])

//Llamando al constructor
Estudiantes s1 = new Estudiantes(1,"Luis");
//Llamando al constructor
Estudiantes s2 = new Estudiantes(2,"Maria");
s1.display();
s2.display();

 Consola:

1 Luis
2 Maria

Diferencia entre constructor y método

  • 1. Los constructores se utilizan para inicializar el estado de un objeto mientras que un método se utiliza para exponer el comportamiento de un objeto

    • 2. El constructor no puede tener tipo de retorno, mientras que el método puede o no tener un tipo de retorno

  • 3. El nombre del constructor debe ser igual que el nombre de la clase, mientras que el nombre del método puede o no ser igual al nombre de la clase

  • 4. El constructor se invoca implícitamente mientras que el método se invoca explícitamente

  • 5. El constructor es proporcionado implícitamente por el compilador java si no especifica ningún constructor en la clase java, pero el método nunca es proporcionado por el compilador java.
