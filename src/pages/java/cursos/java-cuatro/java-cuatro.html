While Loop

Puede haber una situación en la que necesitamos ejecutar un bloque de código varias veces, y es a menudo referido como un bucle

Java tiene tres mecanismo de bucle muy flexibles. Puede utilizar uno de los siguientes tres bucles:

  • 1. While Loop
  • 2. Do... While Loop
  • 3. For Loop

A partir de java 5, se introdujo el bucle for mejorado. Esto se utiliza principalmente para matrices.

While Loop

  • El bucle while repite un segmento de código un número desconocido de veces
  • El código dentro de un bucle while se ejecutará mientras la condición especificada es verdadera

Sintaxis

while (condición es verdadera)

ejecutar el codigo;

Veamos un Ejemplo

class WhileLoop

public static void main(String args[])

int num = 0;

while (num < 25)

num = num + 5;
(num + " " );


 Salida por consola

5 10 15 20 25

En el código anterior, una variable denominada num se inicializa con el valor 0 .

La condición para el bucle while es que mientras num es menor que 25, se debe agregar 5 a num y se debe imprimir (junto con un solo espacio) Una vez que el valor de num sea mayor que 25, el bucle dejará de ejecutarse
