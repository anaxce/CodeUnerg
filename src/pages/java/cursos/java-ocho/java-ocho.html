Métodos

¿Que son los métodos en java?

Un método es un grupo de instrucciones java que realizan alguna operación en algunos datos y pueden o no devolver un resultado

Los métodos java deben estar ubicados dentro de una clase java

Sintaxis

modifier returnType nameOfMethod (Parameter List)

// Cuerpo del metodo
  • modifier: Define el tipo de acceso del método y es opcional utilizar.

  • returnType: El método puede devolver un valor.

  • nameOfMethod: Este es el nombre del metodo. La firma del método consiste en el nombre del método y la lista de parámetros

  • Parameter List: La lista de parámetros, es el tipo, orden y número de parámetros de un método. Estos son opcionales. Método puede contener cero parámetros

  • method body: El cuerpo del método define lo que hace el método con las sentencias

Ejemplo

public static int nombreMetodo (int a, int b)

// Body
  • public static: modifier.

  • int: returnType.

  • nombreMetodo: Nombre del metodo

  • a, b:Parametros formales

  • int a, int b: Lista de parametros (Parameter List)


Modificadores de acceso a métodos y parámetros

Usted puede haber notado la palabra pública en el ejemplo anterior, esta palabra es el modificador de acceso para el método. El modificador de acceso determina qué código se le permite llamar a este método

Parametros del método

  • Parámetros del método hace posible pasar valores al método

  • Los parámetros del método se declaran dentro de los paréntesis después del nombre del método. En el ejemplo siguiente los parámetros del método están marcados en negrita

  • Puede utilizar cualquier tipo de datos primitivo o una clase java incorporada como tipo de datos para los parámetros. También puede utilizar sus propias clases como tipos de parámetros

  • Un parámetro de método es similar a una variable. Usted puede leer su valor, y cambiar su valor también
public void mimetodo (String texto1, String texto2)

System.out.println(texto1);
System.out.println(texto2);

El método mimetodo método en el ejemplo anterior toma dos parámetros llamados texto1 texto2 los dos son de tipo String
