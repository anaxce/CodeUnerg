Tipos de retorno

Tipos de retorno del método

Un metodo en java puede devolver un valor

public int suma(int valor1, int valor2)

return texto1 + texto2;
  • El tipo de retorno es un int(entero), que esta marcado en negrita antes de la suma de nombre del método. Este tipo de retorno indica que este método devuelve un int
  • La instrucción return determina qué valor es devuelto por el método

Ejemplo

public String cuenta (String texto1, String texto2)

return texto1 + texto2;

El método cuenta método en el ejemplo anterior toma dos parámetros llamados texto1 y texto2 de tipo String


Declaraciones de devoluciones múltiples

Es posible tener más de una declaración de retorno en un método. Sin embargo, ambos no se pueden ejecutar. Sólo se puede ejecutar una declaración de devolución. Una vez que se ejecuta una instrucción return, no se ejecuta más código en ese método

Ejemplo

public String cuenta (String string1, String string2, boolean flag)

if (flag)

return string2 + string1;

return string1 + string2;
  • Si el parámetro flag tiene el valor true, entonces el método devuelve las dos cadenas pasadas como parámetros, concatenadas en orden inverso (primera string2, luego string1)

  • Si el parámetro flag tiene el valor false, entonces el método omite el cuerpo de la instrucción if. Entonces continúa a la segunda declaración de retorno.Esta instrucción return devuelve los dos parámetros concatenados en orden normal (string1, y string2)
