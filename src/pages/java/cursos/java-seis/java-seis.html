¿Que es una Clase?

¿Que es una clase en Java?

Una clase es una plantilla de la que se crean los objetos individuales.

Una clase en Java contiene:

  • Campos: Estos campos son variables (datos) que son locales a la clase.

  • Constructores: Constructores son métodos que inicializan una instancia de la clase. Los constructores suelen establecer los valores de los campos en la instancia dada.

  • Metodos: Métodos son operaciones que la clase o instancias de esa clase pueden realizar

Definiendo una clase en Java

Todo lo que se necesita para definir una clase en java es esto

public class Croma

public String croma1 = null;
public String croma2 = null;
public String croma3 = null;

En el ejemplo anterior la clase se crea con el nombre Croma y tiene tres campos
