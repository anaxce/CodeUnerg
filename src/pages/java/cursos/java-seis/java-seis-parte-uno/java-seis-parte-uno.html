Clases con metodos

Vamos a agregar un metodo a la clase Ashe

public class Ashe

public String croma1 = null;
public String croma2 = null;
public String croma3 = null;

public void setColor (String newColor)

this.color = newColor;

En la definición de clase se ha agregado un método llamado setColor(). Cuando se llama, este método establece la variable de color interno (campo) a un nuevo valor.

Clases y Objetos

Para crear objetos de una determinada clase, se utiliza la palabra clave new

Sintaxis

ClassNombre objectNombre = new ClassNombre();

Ejemplo

Croma croma1 = new Croma();
Croma croma2 = new Croma();
Croma croma2 = new Croma();
croma1.setColor("Rosado");
croma2.setColor("Rojo");
croma3.setColor("Azul");

Este ejemplo crea tres variables Croma y asigna una nueva instancia de la clase Croma a cada variable, cada variable ahora hace referencia a un objeto Croma.

Después de crear los tres objetos croma, el método setColor () se llama a cada objeto. Ahora el color (representado como un texto) se fija individualmente para cada objeto de Croma

La creación de un objeto de una determinada clase también se llama instanciar un objeto. Por lo tanto, el objeto también se llama una instancia de la clase dada. Por ejemplo, cada uno de los objetos de croma arriba también se llaman una instancia de la clase de croma, o simplemente las instancias de croma
