Variables

Variable es el nombre del área reservada asignada en la memoria

Ejemplo:

int var = 20; // Aqui var es una variable

Hay tres tipos de variables en Java

  • Variables Locales
  • Variables de Instancia
  • Variables Estaticas

Variables Locales

-Una variable que se declara dentro del método se llama variable local

Variables de Instancia

-Una variable que se declara dentro de la clase pero fuera del método se denomina variable de instancia

Variables Estaticas

-Una variable que se declara como estática se llama variable estática. No puede ser local
