Tipos de Datos en Java

El término tipo de datos se refiere al tipo de datos que se pueden almacenar en una variable

A veces, java se llama un "lenguaje fuertemente mecanografiado" porque cuando yo declare una variable, debe especificar el tipo de la variable

Entonces el compilador asegura que no intente asignar datos del tipo incorrecto a la variable

Hay dos tipos de datos disponibles en java

  • Tipos primitivos
  • Tipos objetos
