Instancia del operador

La instancia de operador se utiliza para determinar si un objeto es una instancia válida de un tipo específico. Se puede usar para probar si un objeto es una instancia de una clase, una instancia de una subclase o una instancia de una clase que implementa una interfaz en particular

Ejemplo

interface Animal



class Mamifero implements Animal




public class Dog extends Mamifero

public static void main(String args[ ])

Mamifero = m new Mamifero();
Perro d = new Perro();

System.out.println(m instanceof Animal);
System.out.println(d instanceof Mamifero);
System.out.println(d instanceof Animal);

 Consola:

true
true
true
