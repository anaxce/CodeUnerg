Herencia Multinivel

En la herencia multinivel, una clase puede heredar de una clase derivada, haciendo de esta clase derivada la clase base para la nueva clase.


Ejemplo

Class A

public void metodoA()

System.out.println("Metodo Clase A");



Class B extends A

public void metodoB()

System.out.println("Metodo Clase B");



Class C extends B

public void metodoC()

System.out.println("Metodo clase C");


public static void main(String args[ ])

C obj = new C();

//Llamando al método de clase abuelo
obj.metodoA();

// llamando al metodo de clase padre
obj.metodoB();

// llamando al metodo local
obj.metodoC();

 Consola:

Metodo Clase A
Metodo Clase B
Metodo Clase C

Herencia Jerarquica

En la herencia jerárquica una clase es heredada por muchas sub clases

En la imagen de abajo la clase b y c hereda la misma clase A. A es la clase principal (o clase base) de B & C



Class A

public void metodoA()

System.out.println("Metodo Clase A");



Class B extends A

public void metodoB()

System.out.println("Metodo Clase B");



Class C extends B

public void metodoC()

System.out.println("Metodo clase C");



Class MyClass

public static void main(String args[ ])

B obj1 = new B();

C obj2 = new C();

//Llamando al método de clase abuelo
obj1.metodoA();

// llamando al metodo de clase padre
obj2.metodoA();


 Consola:

Metodo Clase A
Metodo Clase B
Metodo Clase C
