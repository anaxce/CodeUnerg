Tipos de Herencia

Tipos de Herencia


  • 1) Single: Herencia unica

  • 2) Multilevel: Herencia multinivel

  • 3) Hierarchical: Herencia jerárquica



  • 4) Multiple: Herencia múltiple

  • 5) Hybrid: Herencia Hibrida



La herencia múltiple no es compatible con java. Esto significa que una clase no puede extender más de una clase

Herencia Unica (single)

Cuando una clase extiende una clase se llama como Herencia Unica



Ejemplo

Class A

public void metodoA()

System.out.println("Metodo super clase");



Class B extends A

public void metodoB()

System.out.println("Metodo hijo de clase");


public static void main(String args[ ])

B obj = new B();

//llamando metodo super clase
obj.metodoA();

// llamando al metodo local
obj.metodoB();

 Consola:

Metodo super clase
Metodo hijo de clase
