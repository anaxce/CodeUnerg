¿Que es una Herencia?

  • inheritance can be defined as the process where one class acquires the properties (methods and fields) of another

  • Con el uso de la herencia la información se hace manejable en un orden jerárquico

  • La idea detrás de la herencia en java es que puede crear nuevas clases que se basan en clases existentes. Cuando heredes de una clase existente, puedes reutilizar métodos y campos de la clase padre, y puedes agregar nuevos métodos y campos también

  • La clase que hereda las propiedades de otra clase es conocida como subclase (clase derivada, clase hijo) y la clase cuyas propiedades son heredadas se conoce como superclase (clase base, clase padre)

¿Por qué utilizar la herencia en java?

  • Usar Polimorfismo
  • Para la reutilización del código

Sintaxis

class Subclase-nombre extends Superclase-nombre

// Metodos y campos

Palabra clave extends

La palabra clave extends indica que está creando una nueva clase que hereda las propiedades de una clase existente

  • Vehiculo es la superclase de Moto

  • Moto es una subclase de Vehiculo
