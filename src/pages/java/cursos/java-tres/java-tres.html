Declaración If y If Else

Declaración If

Hay dos tipos de declaraciones de toma de decisiones en java

  • Declaracion If
  • Declracion Switch

Declaración If

Una instrucción if consiste en una expresión booleana seguida de una o más declaraciones

if(Boolean_expression)

//Las sentencias se ejecutarán si la expresión booleana es verdadera.

Si la expresión booleana se evalúa como verdadera entonces se ejecutará el bloque de código dentro de la sentencia if.

Sino se ejecutará el primer conjunto de código después del final de la sentencia if (después del cierre)

public classTestIf

public static voidmain(String args[])

int i = 5;
if( i < 10)

System.out.print("Esto es una declaracion If");

Declaración If Else

Una instrucción if puede ser seguida por una instrucción opcional else, que se ejecuta cuando la expresión booleana es falsa

Ejemplo

if(condición)

//Declaraciones que se ejecutarán si la condición es verdadera

else

// Declaraciones que se ejecutaran si la condición es falsa

Aquí si la condición es verdadera, el código que está escrito dentro de los corchetes del bloque if será ejecutado

Si la condición es falsa, el código que se escribe dentro de los corchetes del bloque else se ejecutará

Realizaremos un pequeño programa

  • Este programa mostrará un mensaje "Éxito" si un valor determinado es mayor que 5
  • Si este valor no es mayor que 5, mostrará un mensaje "fallo"
  • Entonces mostrará el mensaje "ejecutado satisfactoriamente" y completará su ejecución.
//FileName: IfElseDemo1.java

public class IfElseDemo1

public static voidmain(String args[])

//Declarando una variable "nota" e inicializarla con un valor 10
int nota = 10;

Comprobando si "nota" es mayor que 5
if(nota > 5)

//Comprobando si la "prueba" es mayor que 5
System.out.println("Exito");

else

//Este bloque será ejecutado sólo si "prueba" no es mayor de 5
System.out.println("Fallido");


//Final bloque if else
System.out.println("Ejecutado Satisfactoriamente");
