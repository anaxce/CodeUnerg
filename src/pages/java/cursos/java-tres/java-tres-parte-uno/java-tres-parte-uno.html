Declaración Switch

Switch también se usan cuando necesitamos que nuestro programa tome una decisión determinada basada en una condición y luego ejecute en consecuencia

La sentencia switch se encarga de estructurar una selección múltiple. Al contrario del enunciado if-else que sólo podemos indicar dos alternativas, maneja un número finito de posibilidades.

Es mas sencillo de lo que suena, veamos un ejemplo

/* Ejemplo metodo que usa switch */
public void dimeSiEdadEsCritica()

switch(edad)

case 0;
System.out.println("Aun no cumple el primer año");
break;
case 18: System.out.println("Esta justo en la mayoría de edad");
break;
case 65: System.out.println("Está en la edad de jubilación");
break;
default: System.out.println ("La edad no es crítica");
break;

Otro Ejemplo

public class SwitchDemo1


public static void main (String args[])

int eleccion = 2;

switch (eleccion)

case 1:
System.out.println("Menu Banco")
break;

case 2:
System.out.println("Menu Prestamo")
break;

case 3:
System.out.println("Menu Tarjeta de Credito")
break;

default:
System.out.println("No se consigue el resultado")
break;

Menu Prestamo
