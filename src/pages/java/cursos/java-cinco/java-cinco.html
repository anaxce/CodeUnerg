Arrays

Que es un Array? tambien conocida como Matriz

  • Array es una estructura de datos proporcionada por java, que almacena una colección secuencial de tamaño fijo de elementos del mismo tipo

  • Array en java está basado en índices. Cada elemento de una matriz se llama un elemento, y cada elemento se accede por su índice numérico.

  • El primer elemento de la matriz se almacena en 0 índice.

Declarando un Array

Sintaxis

type [ ] arrayNombre;

Ejemplo

// Declara una matriz de enteros
int [ ] anArray;

Las declaraciones de Arrays se dividen en dos partes

  • Tipo de Array: Se escribe como tipo [], donde type es el tipo de datos de los elementos contenidos, los paréntesis son símbolos especiales que indican que esta variable contiene un array
  • Nombre de Array:Puede ser cualquier cosa que quieras, siempre que se siga las reglas y parametros.

De forma similar, puede declarar matrices de otros tipos.

boolean[ ] anArrayBooleans;
String[ ] anArrayStrings;
double[ ] anArrayDoubles;
short[ ] anArrayShorts;
float[ ] anArrayBooleans;
char[ ] anArrayChars;
byte[ ] anArrayBytes;
long[ ] anArrayLongs;

También puede colocar los corchetes después del nombre de la matriz. Sin embargo, la convención desalienta esto, los corchetes identifican el tipo de matriz que debe aparecer con la designación de tipo.

// Esta forma es desalentada
float anArrayFloat [ ];

Puede crear una matriz utilizando el nuevo operador con la siguiente sintaxis

Sintaxis

dataType[ ] arrayNombre = new dataType[arrayTamaño]

La declaración anterior hace dos cosas

  • Crear una nueva matriz usando dataType[arrayTamaño].

  • Asigna la referencia de la matriz recién creada a la variable arrayNombre.
