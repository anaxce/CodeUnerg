Inicializando

Sintaxis

arrayNombre[index] = value;

Las siguientes líneas asignan valores a cada elemento de la matriz

Incializar el primer elemento
anArray[0] = 100;
Incializar el segundo elemento
anArray[1] = 200;
Etcétera
anArray [2 ] = 300;

Declarar una variable de matriz, crear una matriz y asignar la referencia de la matriz a la variable se pueden combinar en una sentencia, como se muestra a continuación.

// crear un array de enteros
int [ ] anArray = new int [10];

Alternativamente, puede crear matrices de la siguiente manera

dataType[ ] anArray = valor0, valor1, ..., valuek ;

Ejemplo

int [] anArray = 100, 200, 300, 400, 500, 600, 700, 800, 900, 1000;

Puede acceder a la longitu(length)d de una matriz a través de su campo de longitud.

Sintaxis

arrayNombre.length

Ejemplo

int[ ] anArray = new int[ 10];
int arrayLength = anArray.length;

En este ejemplo la variable llamada arrayLength contendrá el valor 10
