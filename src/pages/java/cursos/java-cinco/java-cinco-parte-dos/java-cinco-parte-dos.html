Accediendo al Array

Se accede a los elementos de la matriz a través del índice. Los índices de matriz están basados en 0, es decir: comienzan desde 0 hasta una longitud de anArray.length-1

Ejemplo // Imprime el valor del primer elemento
System.out.println(" Elemento 1 en el indice 0: " + anArray[0]);

// Imprime el valor del segundo elemento
System.out.println(" Elemento 2 en el indice 1: " + anArray[1]);

// Imprime el valor del tercer elemento
System.out.println(" Elemento 3 en el indice 2: " + anArray[2]);

// Imprime el valor del ultimo elemento
System.out.println(" Ultimo elemento : " + anArray[anArray.length-1]);
